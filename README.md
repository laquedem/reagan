# REAGAN
----------------------------------------------------------------------------------------------------------------
# What is this?
It is a game in top-down shooter genre developed on Unity 2017. It has not only TDS game mechanics, but also RPG (equipment, inventory, trade) and survival (health decreases, collecting items).
